إعداد Firebase (مجاني)
متزامن فوري بين كل أجهزتك · بياناتك لا تُحذف · مجاني تماماً
خطوات الإعداد (مرة واحدة فقط)
1
أنشئ مشروع Firebase
اذهب إلى console.firebase.google.com ← New Project ← أي اسم
2
فعّل Firestore Database
Build ← Firestore Database ← Create database ← Start in test mode
3
فعّل Storage
Build ← Storage ← Get started ← Start in test mode
4
احصل على Config
Project Settings (⚙) ← Your apps ← أضف Web app (</>) ← انسخ الـ config
الصق إعدادات Firebase
apiKey
authDomain
projectId
storageBucket
messagingSenderId
appId
الإعدادات تُحفظ في متصفحك · كرر الخطوة في كل جهاز جديد
لمشاركة نفس البيانات: فقط استخدم نفس الـ config في كل الأجهزة
